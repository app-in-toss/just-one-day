📄 [프로젝트] 절약 습관 형성 및 보상 앱 (가칭: 하루하나)Requirements Specification v1.01. 서비스 개요목표: 사용자가 일상적인 지출(커피, 택시 등)을 참았을 때 실질적인 보상(포인트)을 제공하여 절약 습관을 형성함.핵심 모델: 절약 실천 확인 + 광고 시청 완료 → 포인트 지급 (리워드형 광고 모델)차별화 요소: 연속 달성(Streak)에 따른 강력한 보상 체계 (7일 / 30일 단위).2. 주요 기능 및 화면 구조 (User Flow)2.1 인트로 및 목표 설정 (Intro & Setup)인트로: 서비스 가치 제안 및 '오늘부터 시작하기' 유도.카테고리 선택: 절약할 항목 선택 (커피, 택시, 담배, 배달, 충동구매 등).단가 설정: 선택한 항목의 1회 평균 지출액 설정 (예: 커피 4,500원).2.2 메인 대시보드 (Main Dashboard)상태 표시: 이번 주 총 절약 금액 합산 노출.주간 캘린더: 월~일 요일별 실천 여부를 점(Dot) 형태로 표시.기록 요약: 현재 연속 기록(Day), 누적 획득 포인트 표시.미래 예측: 현재 페이스 유지 시 '한 달 후 예상 절약액' 계산 노출.2.3 실천 및 보상 (Check-in & Reward)체크인: '오늘 체크 완료' 버튼 클릭 시 광고 팝업 실행.광고 연동: 광고 시청이 완전히 끝난 후 보상 콜백 수신.포인트 지급: 기본 보상 + (조건 충족 시) 연속 보너스 지급.3. 상세 보상 로직 (Reward Logic)기본 보상: 광고 시청 완료 1회당 정해진 기본 포인트 지급.연속 보너스 (Streak Bonus):7일 연속: 7일간 하루도 빠짐없이 체크 성공 시 추가 보너스 지급.30일 연속: 한 달간 완주 시 대형 보너스 지급.예측 알고리즘: (이번 주 평균 절약액) x 4.3 등의 공식을 활용해 월간 기대 수익 시각화.4. 데이터 스키마 (Data Schema)4.1 UsersFieldTypeDescriptionuser_idUUID고유 식별자current_streakInt현재 연속 달성 일수total_pointsInt보유 포인트total_savedInt누적 절약 금액4.2 GoalsFieldTypeDescriptiongoal_idUUID목표 IDcategoryString선택 항목 (Coffee, Taxi 등)unit_priceInt1회 절약 단가4.3 RecordsFieldTypeDescriptionrecord_idUUID기록 IDdateDate실천 날짜ad_statusBoolean광고 시청 완료 여부is_bonusBoolean보너스 지급 여부 (7/30일)
